# 图片通过命令行处理

# 安装

  • 先安装 imagemagick
  // brew install ImageMagick
  ~~~ brew install ImageMagick ~~~
  • 然后安装 imagick
  pecl install imagick

# 具体指令

  • 大小调整：将图像的大小调整为200像素宽和100像素高

      convert demo1.png -resize 200×100 demo2.png

  • 旋转角度：demo1.png 的图像，将其旋转90度，并将旋转后的图像保存为 demo1-rotate.png

      convert demo1.png -rotate 90 howtogeek-demo1.png

  • 滤镜效果：

    • 下面的是‘木炭’风格效果，强度为2
        convert demo1.png -charcoal 2 demo1-charcoal.jpg
    • implode风格效果 ，强度为 1 +
        convert demo1.png -implode 1 demo1-charcoal.jpg

  • 各种组合 +

      convert demo1.png -resize 400×400 -rotate 180 -charcoal 4 -quality 95 demo1-all.png
