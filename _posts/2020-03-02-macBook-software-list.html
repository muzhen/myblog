# 生活类的

  • # 记账软件

    • 随手记

      • 最早用的应该是这个，2014年 出来实习之后开始用，后来花样越来越多，信息越来越杂，然后就不怎么喜欢用这款软件的，也是用到2016年左右，后来想找个可以同步账单的，这个同步数据也不怎么喜欢，要配合卡牛一起才能同步，后来就放弃了，导出数据的时候，发现要是会员才能导出！！！！，所以我导出数据之后立马放弃这个软件

    • 网易有钱

      • 这个是随手记之后，用的第二款软件，什么都喜欢，可以导入随手记的账单，而且ui界面很好看，交互的也友好，客服回答问题也及时，真的是那时的心头好啊，不用每天记账，一个礼拜同步一次数据，重复的会被检测出来，感觉很用心的一款软件；
      • 用到19年10月左右吧，用了两年多，最后败在同步信用卡、储蓄卡、支付宝功能全暂停了，说是因为不安全，建设银行直接提示这款软件不安全，建议卸载；因为这个原因，我一直没找到更加合适自己的记账软件
      • 这款软件导出数据的时候 ，也要是会员！！！！，所以我又交钱了！！！！😭😭😭

    • money pro 💗💗💗💗💗

      • 这个是目前的最爱，由于之前使用的两个软件，导致我特别在意数据的导入导出问题，这款很好，数据的导入导出都是免费的 😁😁😁😁😁
      • 真的是非常推荐的一款记账软件，有mac、windows、iphone、ipad、andriod版本
      • 同步的话 由于我是iphone的手机，自己一个人用的情况就可以通过同一个icloud账户，同步mac和手机的数据
      • 如果是大家一起共同管理一个账户的话，那就要花费一些费用，订阅 PLUS（Money Pro Sync服务）或 GOLD（Money Pro Sync 服务和网上银行）才能同步数据
      • 和大部分的手机软件不同的是，这款可以免费导出数据csv、pdf格式的
      • 导入的话支持 .csv 和 .ofx 格式文件，如果重复数据是会被检测出来
      • 还有我用的比较少的各种报告模版、预算设定和跟踪，也是很赞的！！！
      • 这个软件供应商是 iBear LLc，支持中文，所以不用担心英文不好的问题 😏😏😏
      • 详细介绍，点击这里查看官方的用户手册

    • moneyWiz 💗💗💗💗

      • 这个之所以没选，是因为这个界面风格个人不是很喜欢
      • 目前出了moneyWiz 3,订阅好像是免费的
      • 也是国外的软件，支持中文，银行卡中国的支持的和前面的money pro 一样不多
      • 导入导出也差不多，都是免费的，支持csv格式
      • 也有各种的报告、预算的
      • 功能上面和money pro差不多
